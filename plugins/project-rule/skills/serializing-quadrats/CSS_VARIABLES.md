# Quadrats Serializer CSS Variables Reference

Use Quadrats CSS variables in frontend rendering to maintain visual consistency with the editor.

## Install Theme Styles

```bash
npm install @quadrats/theme
```

```scss
// Import default theme
@import '@quadrats/theme/src/qdr.scss';

// Or import dark theme
@import '@quadrats/theme/src/qdr-dark.scss';
```

---

## Apply Theme

### Method 1: Using ConfigsProvider

```tsx
import { ConfigsProvider } from '@quadrats/react/configs';
import { zhTW } from '@quadrats/locales';

function ArticlePage({ content }) {
  return (
    <ConfigsProvider theme="light" locale={zhTW}>
      {({ theme: { props: { className, style } } }) => (
        <article className={className} style={style}>
          {serializer.serialize(content)}
        </article>
      )}
    </ConfigsProvider>
  );
}
```

### Method 2: Manually Add CSS Class

```html
<article class="qdr-theme-light">
  <!-- Serialized content -->
</article>

<article class="qdr-theme-dark">
  <!-- Dark theme -->
</article>
```

## Component-Specific CSS Variables

### Image

```scss
// Automatically set by element.align
--qdr-image-align: flex-start | center | flex-end;
--qdr-image-caption-align: left | center | right;
```

Custom image renderer usage example:

```tsx
createJsxSerializeImage({
  figure: {
    render: ({ children, style }) => (
      <figure
        className="article-image"
        style={{
          ...style,
          display: 'flex',
          justifyContent: 'var(--qdr-image-align, center)',
        }}
      >
        {children}
      </figure>
    ),
  },
  caption: {
    render: ({ children, isEmpty }) =>
      isEmpty ? null : (
        <figcaption
          style={{ textAlign: 'var(--qdr-image-caption-align, center)' }}
        >
          {children}
        </figcaption>
      ),
  },
});
```

## Custom Style Examples

### Basic Article Styles

```scss
.article-content {
  color: var(--qdr-text-primary);
  line-height: var(--qdr-typography-article-body1-line-height);

  h1, h2, h3, h4, h5, h6 {
    margin-top: var(--qdr-spacing-14);
    margin-bottom: var(--qdr-spacing-7);
  }

  p {
    margin-bottom: var(--qdr-spacing-7);
  }

  blockquote {
    margin: var(--qdr-spacing-9) 0;
    padding: var(--qdr-spacing-7);
    background: var(--qdr-block);
    border-left: 4px solid var(--qdr-primary);
    border-radius: var(--qdr-radius-2);
  }

  hr {
    margin: var(--qdr-spacing-14) 0;
    border: none;
    border-top: 1px solid var(--qdr-divider);
  }

  a {
    color: var(--qdr-primary);
    text-decoration: none;

    &:hover {
      text-decoration: underline;
    }
  }

  img {
    max-width: 100%;
    border-radius: var(--qdr-radius-3);
  }

  figcaption {
    margin-top: var(--qdr-spacing-4);
    color: var(--qdr-text-secondary);
    font-size: var(--qdr-typography-article-body2-font-size);
  }

  table {
    width: 100%;
    border-collapse: collapse;
    margin: var(--qdr-spacing-9) 0;

    th, td {
      padding: var(--qdr-spacing-4) var(--qdr-spacing-6);
      border: 1px solid var(--qdr-border);
    }

    th {
      background: var(--qdr-block);
      font-weight: 600;
    }
  }
}
```

## Tailwind CSS Integration

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        qdr: {
          primary: 'var(--qdr-primary)',
          'text-primary': 'var(--qdr-text-primary)',
          'text-secondary': 'var(--qdr-text-secondary)',
          surface: 'var(--qdr-surface)',
          block: 'var(--qdr-block)',
          border: 'var(--qdr-border)',
          divider: 'var(--qdr-divider)',
        },
      },
    },
  },
};
```

```tsx
// Custom renderer using Tailwind
createJsxSerializeBlockquote({
  render: ({ children }) => (
    <blockquote className="my-6 p-4 bg-qdr-block border-l-4 border-qdr-primary rounded">
      {children}
    </blockquote>
  ),
});
```

## Custom Theme

### Override Variables

```css
.my-article-theme {
  --qdr-primary: #1890ff;
  --qdr-text-primary: #1f2937;
  --qdr-text-secondary: #6b7280;
  --qdr-block: #f3f4f6;
  --qdr-border: #e5e7eb;
  --qdr-divider: #d1d5db;

  /* Custom typography */
  --qdr-typography-article-body1-line-height: 1.8;
}
```

```tsx
<article className="my-article-theme">
  {serializer.serialize(content)}
</article>
```
